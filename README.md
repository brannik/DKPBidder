# DKPBidder Addon

## Overview
DKPBidder is a World of Warcraft addon designed for managing DKP (Dragon Kill Points) bidding in raids. It provides a user-friendly GUI for players to place bids and track their DKP.

## Features
- GUI window for bidding
- Minimap icon for easy access
- Lightweight and efficient

## Installation
1. Download the DKPBidder addon files.
2. Extract the contents into your World of Warcraft addons directory, typically located at `World of Warcraft/Interface/AddOns/`.
3. Ensure the folder structure is maintained:
   ```
   DKPBidder/
   ├── DKPBidder.lua
   ├── DKPBidder.toc
   ├── libs/
   │   └── LibStub/
   │       └── LibStub.lua
   └── media/
       └── icon.tga
   ```
4. Launch World of Warcraft and enable the DKPBidder addon in the AddOns menu.

## Usage
- Click the minimap icon to open or close the DKPBidder window.
- Use the GUI to place bids and view current DKP standings.

## Dependencies
- LibStub: A lightweight library for managing other libraries in WoW addons.
